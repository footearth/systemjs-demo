# Systemjs Demo

## global require

### harp

you must install harp server with global flag

```bash
npm i -g harp
```

## commands

### clean

clean build/dist directorys

```bash
npm run clean
```

### updateLibs

download and overwrite libs like jquery/lodash/...

```bash
npm run updateLibs
```

### start

start harp server

```bash
npm start
```

### build

build jade/coffee to html/js and demploy to build directoy

```bash
npm run build
```
